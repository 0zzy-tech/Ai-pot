ABOUT

AI Honeypot is a lightweight deception platform that simultaneously masquerades as 13 AI API servers — Ollama, OpenAI, Anthropic, HuggingFace, Stable Diffusion, and more — on a single port. Every request is captured, classified, and enriched with geo, ISP, reverse DNS, AbuseIPDB reputation, GreyNoise classification, and threat feed lookups (Feodo C2 & ThreatFox IOCs) in real time.

Built to answer a simple question: what actually hits an exposed AI endpoint on the internet? Designed to run on a Raspberry Pi with no cloud dependencies — a single async worker, SQLite, and ~384 MB RAM.

Built by Martyn Oswald — 0zzy-tech  •  github.com/0zzy-tech/Ai-pot
